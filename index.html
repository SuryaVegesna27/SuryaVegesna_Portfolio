Projects

Yelp Review Sentiment Analysis

This project involves developing a scalable sentiment analysis system using Yelp review data to understand customer sentiment toward businesses.

  • Developed a data processing pipeline on GCP Cluster using Apache Spark and RDDs to preprocess and analyze 3 million Yelp reviews, employing word clouds to highlight key sentiment terms.
  • Led data cleaning initiative, refining a dataset of 3 million Yelp reviews with Apache Spark for accurate sentiment analysis.
  • Optimized Logistic Regression and Naive Bayes models, achieving over 85% accuracy in classifying sentiments within Yelp reviews.
  • Analyzed and visualized sentiment trends using NLP, revealing patterns across 633,771 Yelp users with detailed metrics.
  • Tools: Python, Apache Spark, GCP, NLP.

Automated Tumor Detection using Deep Learning

This project aims to develop an automated deep learning-based system for early detection and localization of brain tumors in MRI scans.

  • Utilized DenseNet201 in TensorFlow and Keras for automatic brain tumor detection, achieving 97-98% precision and 95-99% recall.
  • Orchestrated a dataset with 34.95% tumor images, using ImageDataGenerator for rescaling, segmentation mask application, and dataset splitting.
  • Implemented ResUNet for precise tumor localization and achieved consistent model performance with weighted averages for precision, recall, and F1-score at 0.97, ensuring reliability in medical diagnostics.
  • Tools: Python, TensorFlow, Keras, Deep Learning.

Tesla Stock Price Prediction

This project explores the use of ARIMA, SARIMA, and LSTM neural networks to predict the opening stock prices of Tesla Inc., aiming to enhance the predictive accuracy and reliability of financial market forecasts.

  • Utilized ARIMA and SARIMA models for capturing linear relationships and seasonality, while employing LSTM networks for understanding long-term dependencies in sequential data.
  • Engineered features and normalized data using MinMaxScaler, essential for the LSTM model to optimize network convergence during training.
  • Trained and evaluated models, achieving a Root Mean Squared Error (RMSE) of 0.036 for ARIMA and 4.02 for LSTM, indicating high predictive accuracy.
  • Forecasted stock prices and visualized results, revealing potential upward trends despite recent downturns, suggesting a resilient growth outlook for Tesla's stock.
  • Tools: Python, TensorFlow, Keras, ARIMA, SARIMA, LSTM.

Telecom Customer Churn Prediction

This project aims to build a predictive model to address the issue of high customer churn in a telecommunications company. The model uses machine learning techniques to predict whether a customer is likely to churn based on features such as contract type, tenure, monthly charges, internet service type, and customer demographics.

  • Engineered and preprocessed features from raw customer data, including transaction history, engagement metrics, and support interactions, to create a robust dataset for model training.
  • Conducted exploratory data analysis to identify key patterns, such as higher churn rates among customers with month-to-month contracts and those without dependents.
  • Developed logistic regression and random forest models to predict customer churn, achieving an accuracy of 80% with logistic regression and comparable performance with random forest.
  • Analyzed model performance, revealing that longer contracts and DSL internet service reduce churn likelihood, while higher total charges and fiber optic internet increase churn risk.
  • Achieved consistent cross-validation scores, with logistic regression showing slightly better recall than random forest, indicating its suitability for churn prediction.
  • Tools: Python, Jupyter Notebook, scikit-learn.
